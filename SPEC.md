# Flickr 写真ダウンローダー 仕様書

## 概要

Flickr で管理している写真を、年月単位でローカルにダウンロードするための Web アプリ。
自分1人が使うツールとして動作する。

---

## 利用環境

- ブラウザで操作する Web アプリ
- 単一ユーザー向け（自分のアカウントの写真のみ取得）

---

## 認証

- Flickr の OAuth を使って認証する
- API キーなどの認証情報はコードに埋め込まず、設定ファイルや環境変数で管理する

---

## 機能

### 1. 期間指定

以下の2通りの指定方法に対応する。

| 指定方法 | 説明 | 例 |
|----------|------|----|
| 単月指定 | 1ヶ月分をダウンロード | 2024年1月 |
| 範囲指定 | 複数月をまとめてダウンロード | 2023年1月 〜 2024年3月 |

### 2. 写真サイズの選択

ダウンロードする写真のサイズを選択できる。
Flickr が提供するサイズ（小・中・大・オリジナル など）から選べるようにする。

### 3. ダウンロード先フォルダの選択

ダウンロード先フォルダを毎回指定できる。

### 4. ダウンロード実行

指定した期間・サイズ・フォルダの設定でダウンロードを実行する。

---

## ダウンロード後のフォルダ構成

```
<指定フォルダ>/
  2024/
    01/
      photo1.jpg
      photo2.jpg
    02/
      photo3.jpg
  2023/
    12/
      photo4.jpg
```

---

## 対象外

- メタデータ（タイトル・タグ・説明など）の保存は行わない
- 他ユーザーの写真へのアクセスは対象外
